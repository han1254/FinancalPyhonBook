1.1 基本的数据类型

1.1 基本的数据类型

1 数值（int float True/False）

记住，python中你几乎不用直接指出一个变量是什么类型，计算机会通过你给这个变量赋的值来自动判断它是什么类型。

比如，我设置一个变量叫n1，n1是什么类型呢？不知道；但是如果我给它一个值，比如3，即n1 = 3，那么我们就能直到，哦，原来n1是个数值。

那么数值也分很多种，有整型（就是你平常理解的整数）(int)、浮点型（你可以理解为带小数点的数）(float)、布尔型（bool，也就是True或者False，本身他也就是1和0）

2 字符串（str）

顾名思义，就是串字符，简单来说，我现在说的一句话就是一个字符串。但是在python中，要把一个字符串用''来包含住。

str1 = 'Hello beginer!'

那么现在我们来更新一下思维导图

3 列表（list）

列表是一种常见的数据类型，比如下面这个，我们看报表种类一栏，就有——基本报表，往来报表，资金报表等等，所以，我们需要一种数据类型，能够存储一列数据，那么这种类型就叫做列表。

可以自己试一试，看看能不能输出东西。

这时候你可能有疑问了？这个里面是只能存一类数据吗？No，python这么语言就是这么强大，它很自由，里面啥数据能放。

看，我把之前放的字符串换成一个数字3，一个True，也是没问题的。

4 元组（tuple）

用法与列表大致相同，不过是要用小括号括起来。

5 集合（set）

用法与列表大致相同，不过是要用大括号括起来

6 字典（dictionary）

一说起字典，你是不是就想起来《新华字典》了？

其实有个更加生动的例子：

用浏览器搜索谷歌翻译，我输入一个英文，就可以得到一个对应的中文。比如我用'beautiful'就能获得'美丽'这个词，左边的词语就像钥匙key一样，打开对应的门，所以我们称左边为键（key），右边为值（value）。

添加第二个例子仍然是要提示你，字典，也是啥都能往里加的。

我们更新一下思维导图，把列表、元组、集合、字典放到一起看（因为这几个很容易搞混。我刚认识他们的时候也是这个感觉，很乱。）

首先我们来分析一下特征：列表、元组、集合，看起来跟亲兄弟一样，几乎长得一样——列表用'[]'，元组用'()'，集合用'{}'，

记住，

  • 列表是可变的，我可以添加、删除里面的元素
  • 元组是不可变的，不能往里面添加、也不能往里面删除
  • 集合是可变的，可以添加删除元素，但是里面的元素不能重复，重复的话就不往里面添加了

记不住？我们先来看看实例。

首先我们先把这几个变量都打印出来

代码

list1 = [1, 2, 'Three', True]#列表

tuple1 = (1, 2, 'Three', True)#元组

set1 = {1, 2, 'Three', True}#集合

print(list1)
print(tuple1)
print(set1)

等一下，好像哪里不对劲？

我set1里面设置的值明明是1,2,'Three',True，咋输出的时候只有1, 2, 'Three'了？

如果你像我一样在这里发生了疑问，可以看看第一节，我们在讲数值的时候怎么说的？

原来True或者False是可以当作1和0两个数值来看待的。而且我们也说了，set内的值是不会重复的，因为我们之前已经有一个1了，所以再添加一个True，他也只含有一个1，不会再添加一个1了。不信你添加一个False试试。

代码

list1 = [1, 2, 'Three', True]#列表

tuple1 = (1, 2, 'Three', True)#元组

set1 = {1, 2, 'Three', False}#集合

print(list1)
print(tuple1)
print(set1)

继续回归正题，我们来看看列表、元组、集合的不同点（这些了解一下就行，下一节也有）：

好了，到这里基本数据类型应该已经差不多了。字典那个应该挺好记的，跟这三个看起来也不太像。

这三个一个用[]，一个用()，一个用{}，你可以这样记：

列表：你平时用的表格都是方方正正的，所以列表用方括号。

元组：都叫元组了，一定要用圆形的小括号。

集合：集合看起来很花心（花括号），但是内心很专一，已经有一个，就不会接受另一个。
